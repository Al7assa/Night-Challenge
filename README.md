### About The Project

**Night Challenge** is a cyberpunk-style login challenge.  
The goal is simple: **Login successfully** and claim your victory.


---

### How To Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Al7assa/Night-Challenge.git
   cd Night-Challenge
   ```
2. Open the project using Live Server
